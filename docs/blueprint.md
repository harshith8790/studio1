# **App Name**: Content Spark

## Core Features:

- User Onboarding: Collect user data including platform, content niche, and growth goals to personalize the app experience.
- AI Content Calendar: Generate a content calendar tailored to the user's niche, platform, and goals, providing a structured posting schedule. It reasons with the tool to automatically integrate trends.
- Caption and Hashtag Generator: Generate relevant captions and hashtags based on the user's input, niche, platform, and audience size. Provide explanations for the AI suggestions to enhance user understanding and trust. The output includes strategic rationale, platform-optimized text, and hashtags
- Trend Radar: Display trending topics within the user's niche, highlighting low-competition opportunities. The trend will include a strategic rational, and 5 relevant hashtags.
- Analytics Dashboard: Display key engagement and reach insights, including an AI-driven analysis of post performance with explanations for successes and failures.
- Collaboration Suggestions: Suggest content creators with similar niches and follower ranges for potential collaborations and create a simple collaboration request workflow.
- Content Logs: Store all generated content for the users and save them in a firestore collection.

## Style Guidelines:

- Primary color: Vivid purple (#9D4EDD) for a modern, creative feel.
- Background color: Light lavender (#E9D5FF) to create a soft and clean backdrop.
- Accent color: Electric blue (#50C5FF) to highlight interactive elements and important information.
- Body and headline font: 'Inter' for a clean and modern look across all text elements.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use consistent, minimalist icons to represent different features and data points.
- Implement a clean 'SaaS-style' UI with a sidebar for easy navigation and clear content presentation, complete with platform selectors and mockup cards for realistic output.
- Add subtle transitions and animations to create a dynamic user experience and enhance the feeling of intelligence and support.
